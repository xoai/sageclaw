# Productivity Template

A team of 3 agents for personal productivity:

- **Coordinator** (fast) — orchestrates tasks, delegates, synthesizes results
- **Researcher** (strong) — web research, data analysis, knowledge building
- **Coder** (strong) — code writing, review, debugging

## Quick Start

```bash
sageclaw init --template productivity
sageclaw
```

Or apply from the dashboard: Settings > Templates > Productivity > Apply

## How It Works

1. You message the Coordinator
2. Coordinator breaks your request into subtasks
3. Researcher and Coder work on assigned tasks in parallel
4. Coordinator synthesizes results and responds

## Example Requests

- "Research the best Go testing frameworks and set up tests for our project"
- "Find information about the new tax rules and create a summary document"
- "Analyze our API endpoints and suggest performance improvements"
