# Researcher Template

A 2-agent team for deep research:

- **Analyst** (strong, lead) — synthesizes findings, builds knowledge graphs, produces reports
- **Explorer** (fast) — discovers information via web search

## Quick Start

```bash
sageclaw init --template researcher
sageclaw
```

## Example Requests

- "Research the state of AI regulation in the EU — key laws, timeline, impact"
- "Compare the top 5 project management tools for small teams"
- "What are the emerging trends in renewable energy storage?"
