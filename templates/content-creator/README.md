# Content Creator Template

A 3-agent team for content creation:

- **Editor** (strong, lead) — manages pipeline, reviews quality, delivers polished content
- **Writer** (strong) — drafts articles, blog posts, newsletters, social media
- **SEO Specialist** (fast) — keyword research, title optimization, readability analysis

## Quick Start

```bash
sageclaw init --template content-creator
sageclaw
```

## Example Requests

- "Write a blog post about the future of remote work — 1500 words, professional tone"
- "Create a newsletter announcing our new product launch"
- "Draft 5 social media posts for our upcoming webinar"
