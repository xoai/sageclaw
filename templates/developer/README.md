# Developer Template

A 3-agent team for software development:

- **Architect** (strong, lead) — designs systems, creates tasks, makes technical decisions
- **Coder** (strong) — implements features, writes tests, fixes bugs
- **Reviewer** (strong) — reviews code for bugs, security, and maintainability

## Quick Start

```bash
sageclaw init --template developer
sageclaw
```

## Example Requests

- "Build a REST API for user management with CRUD operations"
- "Refactor the payment module to use the strategy pattern"
- "Fix the race condition in the session handler and add tests"
